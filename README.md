# 基于react + and design实现的权限管理后台

- [项目演示](http://admin.adiandian.cn)
- [React](https://react.docschina.org/)
- [Ant Design](https://ant.design/docs/react/introduce-cn)

```
git clone git@github.com:EcoIris/eco-auth-admin.git
```
安装、初始化和编译使用npm或yarn
##npm
```
npm install 

npm start 

npm run build 
```
##yarn
```
yarn install 

yarn start 

yarn run build 
```
